# To-Do List Uygulaması

Bu uygulama, basit bir görev listesi (To-Do List) uygulamasıdır. Kullanıcılar görevlerini ekleyebilir, silebilir ve tamamlanmış olarak işaretleyebilirler.

## Kurulum

1. Python 3.9+ yüklü olduğundan emin olun.
2. Gerekli kütüphaneleri yüklemek için:
    ```
    pip install -r requirements.txt
    ```

## Kullanım

1. Uygulamayı başlatmak için:
    ```
    python main.py
    ```
